# Anonymeet

Rjesavanje problema: Manjak spontanih društevnih interakcija zbog pretjerane upotrebe društvenih mreža. Napraviti mrežu koja te čini društenim.

Koncipira se na tome da user bude i vodič i bude vođen u shopping, birtije, izlazak, turu po gradu...

Slicno kao guruWalk, ali s manje formalnosti u obilasku. Svaka osoba kad izradjuje profil, temeljito ga se ispituje o njegovim interesima i kako voli provesti svoj dan. Hobby i te gluposti. Bazirano o navedenim interesima po profilima drugi user bira zeli li provesti vrijeme s tom osobom.
